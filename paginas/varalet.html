Variables aleatorias

Las variables aleatorias, tanto discretas como continuas, se definen mediante distribuciones de probabilidad que permiten cuantificar la incertidumbre de los posibles valores que pueden tomar, a través de medidas como el valor esperado, la varianza y desviación estándar, así como la función de distribución acumulada, la cual facilita el cálculo de probabilidades; estos conceptos y propiedades de las variables aleatorias constituyen una extensión natural de los fundamentos de la teoría de probabilidad, permitiendo modelar de manera más completa los fenómenos aleatorios.


Número de Subtema Nombre del Subtema Descripción
3.1 Variables aleatorias discretas Este subtema se enfoca en el estudio de variables aleatorias que pueden tomar un número finito o contable de valores.
3.1.1 Distribución de probabilidad en forma general Explica cómo se define y se representa la distribución de probabilidad de una variable aleatoria discreta.
3.1.2 Valor esperado Describe el concepto de valor esperado, que representa el promedio o la media de la variable aleatoria discreta.
3.1.3 Variancia, desviación estándar Introduce las medidas de dispersión, como la variancia y la desviación estándar, que cuantifican la variabilidad de la variable aleatoria discreta.
3.1.4 Función acumulada Explica la función de distribución acumulada, que representa la probabilidad de que la variable aleatoria discreta tome un valor menor o igual a un valor específico.
3.2 Variables aleatorias continuas Este subtema se enfoca en el estudio de variables aleatorias que pueden tomar cualquier valor dentro de un rango de valores.
3.2.1 Distribución de probabilidad en forma general Describe cómo se define y se representa la distribución de probabilidad de una variable aleatoria continua.
3.2.2 Valor esperado Explica el concepto de valor esperado para variables aleatorias continuas, que representa el promedio o la media de la variable.
3.2.3 Variancia, desviación estándar Introduce las medidas de dispersión, como la variancia y la desviación estándar, para variables aleatorias continuas.
3.2.4 Función acumulada Explica la función de distribución acumulada para variables aleatorias continuas, que representa la probabilidad de que la variable tome un valor menor o igual a un valor específico.
3.2.5 Cálculos de probabilidad Describe cómo se realizan los cálculos de probabilidad para variables aleatorias continuas.
